# Hotel Booking Demand Analysis 🏨🐍

## Project Overview
This project performs a comprehensive **Exploratory Data Analysis (EDA)** using Python on a hotel booking dataset. The goal is to understand customer behavior, booking patterns, and identify key factors that lead to booking cancellations.

## Key Insights & Questions Addressed
- **Cancellation Rates:** What are the main factors behind booking cancellations (`is_canceled`)?
- **Lead Time Impact:** How does the time between booking and arrival (`lead_time`) affect the cancellation probability?
- **Seasonality:** Which months have the highest demand for Resort vs. City hotels?
- **Customer Demographics:** Analysis of guests (adults, children, babies) and their meal preferences (`BB`, `HB`, etc.).

## Libraries Used
- **Pandas & NumPy:** For data cleaning and manipulation.
- **Matplotlib & Seaborn:** For creating insightful visualizations (Heatmaps, Bar charts, Time-series).
- **Scikit-learn:** (Optional) If you are planning to build a prediction model.

## Project Structure
- `Hotel_Booking_Analysis.ipynb`: The main Jupyter Notebook containing the code and visualizations.
- `hotel_bookings.csv`: The dataset used for analysis.
- `Screenshots/`: A folder containing key charts generated from the analysis.
